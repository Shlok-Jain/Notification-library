Notification-library : By Shlok Jain

This is a javascript library for showing notifications in javascript apps, React apps and many more..... You can view a demo of it:

Usage :

HTML:
<head>
    <script src="https://cdn.jsdelivr.net/gh/Shlok-Jain/Notification-library@latest/index.js"></script>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Shlok-Jain/Notification-library@latest/index.css">
</head>
JS:
document.getElementById('demo-btn').onclick = () => {
      const my_notification = new Notification('This is an example notification')  //Specify your message here
      my_notification.show(5000)           //write the time for notification in milliseconds
}
